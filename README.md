# MedLine Healthcare Security Program

This repository contains a full, end-to-end cybersecurity and GRC program built for a fictional urgent care clinic called MedLine.

The project demonstrates healthcare security design across:
- HIPAA-aligned risk assessment
- Threat modeling (STRIDE + attack paths)
- Incident response planning and simulations
- Vulnerability management workflows
- Security policy development

## Repository Structure
- 00-overview: Project scope, assumptions, and system description
- 01-risk-assessment: HIPAA-oriented risk assessment artifacts
- 02-threat-modeling: Threat modeling and attack path analysis
- 03-incident-response: IR plan, SIEM-based incident simulation, lessons learned
- 04-vulnerability-management: Vulnerability lifecycle and governance workflow
- 05-policy-suite: Audit-ready security policies
- 06-compliance-mapping: Framework and regulatory mappings
- 99-appendix: Glossary, references, and disclaimers

## Notes
- This is a fictional organization.
- No real patient data (PHI) is included.
- Documents are written for portfolio and educational purposes.
